# Data Directory

This directory contains the data files used in the MCP workshop.

customers.csv is some synthetic data prepared by [Alex Au](https://github.com/alex-au-922) for agentic workflow workshop in April 2025.

image_data.zip contains some images downloaded from Kaggle, you may find the link to the dataset at the end of this README.md.
We use this dataset for the second MCP Server demo.

https://www.kaggle.com/datasets/heavensky/image-dataset-for-unsupervised-clustering
