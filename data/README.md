# Data Directory

This directory contains the data files for the MCP Workshop organized by the Hong Kong Python User Group in June 2025,
which took place in Auki Labs, Hong Kong.

## Synthetic Customer Data

customers.csv is some synthetic data prepared by [Alex Au](https://github.com/alex-au-922) for agentic workflow workshop in April 2025.
You may find the data file at [alex-au-922/HKPUG-Meetup85-AgenticWorkflow](https://github.com/alex-au-922/HKPUG-Meetup85-AgenticWorkflow).
This data is used in the first part of the workshop to demonstrate the use of MCP server developed by others.

## Open-source Image Data

image_data.zip contains some images downloaded from Kaggle.
You may find the dataset at [Kaggle/image-dataset-for-unsupervised-clustering](https://www.kaggle.com/datasets/heavensky/image-dataset-for-unsupervised-clustering).
This data is used in the second and third parts of the workshop to demonstrate some basic setup of self-hosted MCP servers and how to use them to extend the functionality of LLMs.
